- **Python 3.8 or higher** (Python 3.12 recommended)
- **pip** (Python package manager)

- ### 1. Install Required Dependencies

Install all required Python packages using pip:

```bash
pip install streamlit sentence-transformers faiss-cpu numpy
```

# ▶️ Running the Application

### Method 1: Using Python Module (Recommended)

This method works on all systems and is the most reliable:

```bash
python -m streamlit run app_streamlit.py
```

### Method 2: Using Streamlit Command

If `streamlit` is in your system PATH:

```bash
streamlit run app_streamlit.py
```
